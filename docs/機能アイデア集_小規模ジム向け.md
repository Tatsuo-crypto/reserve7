## Fable5のおすすめ順（もし作るなら）

1. **打刻＋勤務時間集計**（1章） — シフト管理の資産にそのまま乗り、あなた自身が毎月使う。決算時の人件費資料にもなる
2. **来店実績＋幽霊会員アラート**（3章） — 予約データから自動で作れて、退会防止＝「継続率」の営業トークに直結
3. **資料ライブラリ**（5章） — 作りは軽いのに「指導の質」が目に見えて上がる
4. **月次売上レポート＋会計CSV**（2章） — 決算前に1回作れば毎年ラクになる
5. **食事へのコメント返し**（4章） — reserve7の差別化の核（ダイエット指導）をさらに強くする

**注意（Z章の原則）**: 全部は作らないこと。選ぶ基準は「①自分のジムで今月困っていること ②商談で相手に言われたこと」の2つだけ。売るために先回りして作るのは、しがない戦略では過剰投資です。
