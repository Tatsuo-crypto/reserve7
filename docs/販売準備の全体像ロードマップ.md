パーソナルトレーナーであるあなたが、開発・販売の経験ゼロから他のジムにreserve7を売るまでの全手順です。**5つのステップを順番にやれば売れる状態になります**。目安は合計1〜1.5ヶ月。

※法律に関わる部分は、私は弁護士ではないので必ず専門家のチェックを受けてください（どこで頼むかも書いてあります）。

## ステップ1: 守りを固める（最優先・数日）

**なぜ最初か**: 売る売らない以前に、今のリポジトリには実際の会員さんの個人情報とパスワード類が入っています。これは自分のジムの問題としてすぐ直すべきものです。

- [ ] 会員さんの実名入りCSV/Numbersファイルを削除（**過去の履歴からも消す** — 履歴に残っていたら消していないのと同じ。作業はSonnetに指示すればできます）
- [ ] パスワード・鍵の全とっかえ（Supabaseのキー、NEXTAUTH_SECRET、Google鍵、VAPID鍵）
- [ ] 平文で書かれたパスワード辞書をハッシュ化
- [ ] ジム名・メールアドレスの直書き（30箇所以上）を設定ファイルに追い出す

**あなたがやること**: Sonnetに「W-4のセキュリティ是正をやって」と指示するだけ。技術作業は全部AIができます。

---

## ステップ2: 法律と書類を揃える（2〜3週間）

**知っておくべき法律は実質3つだけ**です。深く勉強する必要はなく、「何をしてはいけないか」だけ覚えてください。

| 法律 | あなたに関係すること | 対策 |
|---|---|---|
| 個人情報保護法 | 会員の体重・食事記録は大事な個人情報。扱い方の説明責任がある | プライバシーポリシーを用意し、データの削除・返却ができる状態にしておく |
| 景品表示法 | 「売上が倍になる」等、根拠のない効果の断定は違法になりうる | 断定しない。「相手の数字で一緒に計算する」営業に徹する |
| 契約の基本（民法） | 売った物に重大な欠陥があれば責任を問われうる。口約束はトラブルの元 | 契約書に保守範囲・免責を明記する |

**用意する書類は4点**:

1. **利用規約**（アプリを使うルール）
2. **プライバシーポリシー**（個人情報の扱いの説明）
3. **売買契約書**（一番大事。以下を必ず入れる: 価格／セットアップ代行の範囲／サポートは納品後30日まで／以降はスポット料金表（1件2万円〜）／機能追加は対象外／データの削除・返却／効果を保証しないこと）
4. **買い手側に用意してもらうもの一覧**（メールアドレス・クレジットカード・月々のインフラ代5,000〜7,000円がかかること・オプションを使うならGoogleアカウント）

**進め方**: ①Sonnet/Fable5に雛形を作らせる（無料）→ ②弁護士にレビューを頼む。頼み先は「IT・SaaS契約に強い弁護士」で、スポット相談なら**5〜15万円程度**が相場です。地域の商工会議所の無料法律相談で入口を聞くのも手。**この弁護士費用が、今回の販売準備でほぼ唯一の現金支出です**（150万円の売上計画に対する必要経費と考えてください）。

**やってはいけないこと6箇条**（これだけは暗記）:
①契約書なしで納品しない ②ジム同士の会員データを比較する機能を作らない ③「痩せる」「売上が倍」と断定しない ④GoogleとSupabaseの規約を確認してから売る ⑤データを消せない状態で売らない ⑥2〜3社を超えたら法人化と保険を検討する

## ステップ3: 商品を「渡せる形」に整える（2〜3週間）

- [ ] **最小構成の確定**: 基本15万円（予約＋ダイエット記録＋週間パネル＋目標管理＋通知、セットアップ込み）。オプション各2万円（写真自動読み取り／Googleカレンダー同期／オンラインレッスン／複数店舗）、全部盛り20万円
- [ ] **納品手順書とチェックリスト**: 1社目の納品を録画するつもりで手順化。2社目から2〜3時間で終わるように
- [ ] **デモテナント**: サンプルデータ入りのデモ用ジムを1つ（実際の会員さんのデータでは絶対にデモしない）
- [ ] **営業道具2つ**: ROI試算シート（相手の会員数・単価を入れると「月1人の退会を防げば元が取れる」が見える簡単な表）と、自社の実績ダッシュボード（記録継続率・在籍継続率など）

**あなたがやること**: 価格の最終決定と、デモの練習（後述の4分デモ）。技術作業はSonnetへ。

---

## ステップ4: 売る（1ヶ月以内に1社目の商談）

**誰に**: ①知人のジムオーナー（1〜3社目はここ）→ ②導入事例を使った紹介＋SNSでの実演投稿 → ③器具業者・トレーナー養成スクールなど開業者が通る場所（7社目以降）

**何と言って**: 「自分のジムで毎日使っている予約とダイエット指導のアプリ。**月に1人の退会を防げれば元が取れる**」。開発の話ではなく運用の話をする。「同じ規模のジムを自分で回すために作った」は大手にはない武器なので隠さない。

**1社目の特別条件**: 「先着3社限定の特価」にして、代わりに**事例として名前を出させてもらう・導入効果を計測させてもらう**ことを契約に書く（ただの値引きにしない）。

**デモは4分**: 会員がスマホで記録する画面（1分）→ トレーナーが週間パネルで指導する画面（3分）。食事写真の自動読み取りは必ず見せる（オプション販売につながる）。

**商談→契約の流れ**: デモ → ROI試算シートを一緒に入力 → 「買い手側に用意してもらうもの一覧」を渡す → 契約書にサイン → セットアップ日程を決める。

---

## ステップ5: 納品する → 次につなげる

**納品は全部オンラインでできます**: ①ビデオ通話30分で買い手に自分名義のアカウントを作ってもらう（Supabase・Vercel・カード登録）② あなたを一時的にメンバー招待してもらう ③ あなたが遠隔で全部設定（半日、慣れれば2〜3時間）④ 動作確認して引き渡し ⑤ アクセス権を外してもらう。**アカウントは必ず初日から買い手名義**（あなた名義で作って譲渡、は絶対にしない）。

- 納品後30日はサポート、以降はスポット料金表どおり。**「ちょっとした改修」を無償で受けない** — ここが「売って終わり」を守る生命線
- 納品したら: 事例として紹介記事/SNS投稿 → 次の紹介につなげる → 導入効果のデータを集めて営業道具を強くする
