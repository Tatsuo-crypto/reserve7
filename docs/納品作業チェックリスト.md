# 納品作業チェックリスト（1社あたり・AA章反映版）

Gemini化・URL手貼り化・設定の外部化が完了した後の、**あなたが1社ごとに実際にやる作業の全リスト**です。
初回は半日、慣れれば合計2〜3時間。

---

## 事前準備（買い手と会う前・15〜30分）

- [ ] ジム名・ロゴ・ブランドカラーを買い手からもらっておく（LINEでもらうだけでOK）
- [ ] 「買い手側に用意してもらうもの一覧」を事前に送っておく（メールアドレス／クレジットカード（Supabase・Vercel用）／Googleアカウント／独自ドメイン希望の有無／月々のインフラ代5,000〜7,000円の了承）

## ビデオ通話①: アカウント作成の付き添い（30分・買い手と一緒に）

買い手に自分の名義で作ってもらう。あなたは画面共有で案内するだけ:

- [ ] Vercelアカウント作成（Pro契約・カード登録）
- [ ] Supabaseアカウント作成（Pro契約・カード登録）
- [ ] Gemini APIキー発行（Google AI Studioで数クリック・**クレカ不要**）→ キーをあなたに共有
- [ ] SupabaseとVercelにあなたを「メンバー招待」してもらう

## 遠隔セットアップ（あなた1人で・1.5〜2時間）

- [ ] Supabase: 新規プロジェクト作成 → マイグレーション一括実行（手順書どおりコマンドを流すだけ）
- [ ] シークレット新規発行（NEXTAUTH_SECRET、VAPID鍵。**前の顧客の使い回し厳禁**）
- [ ] 環境変数一式をVercelに設定（envテンプレートを埋める: Supabaseキー／Geminiキー／ジム名等）
- [ ] Vercelにデプロイ＋ドメイン設定（買い手の独自ドメイン or Vercelの無料ドメイン）
- [ ] 管理者アカウントを買い手のメールで作成
- [ ] 店舗情報の初期設定（店名・営業時間・予約枠・料金メニュー）
- [ ] 動作確認チェック（予約作成→カレンダー表示／会員登録→トークンログイン／食事写真とスクショの解析／週間パネル表示／プッシュ通知が買い手のスマホに届くか）

## オプションを買われた場合のみ（＋30〜60分）

- [ ] Googleカレンダー同期（2万円）: Google Cloudプロジェクト作成→Calendar API有効化→サービスアカウント作成→カレンダー共有→env設定 ※クレカ不要
- [ ] 複数店舗（2万円）: 2店舗目の店舗設定・予約枠

## ビデオ通話②: 引き渡し（30〜60分・買い手と一緒に)

- [ ] 操作説明（管理画面の使い方／会員の招待のしかた／よくある質問）※説明動画を一度作れば毎回短縮できる
- [ ] 既存会員の登録方法を案内（買い手自身にやってもらう。代行するなら別料金）
- [ ] あなたのメンバー権限を外してもらう（保守契約ありなら残す）
- [ ] 「サポートは今日から30日。以降はスポット料金表のとおり」を口頭でも確認
- [ ] 請求書を送る

---

## あなたがやらなくていいこと（混同注意）

- コードを書く・直す → Sonnetの仕事（納品用ブランチの準備は1回だけ）
- 買い手名義のアカウントをあなたが作る → **禁止**（必ず本人に作らせる）
- 会員データの手入力代行 → 基本は買い手の仕事（やるなら別料金）
- 納品後の「ちょっとした改修」→ 無償で受けない。スポット料金表へ

## 1回だけ作れば全顧客で使い回せるもの（初回前に用意）

- envテンプレート／マイグレーション手順書／動作確認チェックリスト（この紙）／操作説明動画／買い手に渡す一覧表／契約書・請求書の雛形
