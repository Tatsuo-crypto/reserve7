# Reserve7 ダイエット詳細ページ 再設計依頼メモ（Sonnet下調べ・Fable5への相談）

対象: `admin/diet-plan`（トレーナーが会員ごとのダイエット状況を見る画面）。H章・I章の実装（週次サマリー刷新、サマリータブのドメイン分割、週間まとめページ追加）を経てもなお、オーナーから「見やすいと思えない」との評価。ページ構成そのものをFable5に再設計してほしい。

---

## 1. 現状のタブ構成（2026-07-05時点）

食事・栄養 / 体重・生活 / 週間まとめ / 目標設定 の4タブ。「食事・栄養」「体重・生活」はI章の決定でドメイン別に分割したもの、「週間まとめ」は直近の追加要望で作った、体重＋食事＋生活を1ページにまとめたビュー。3つのビューが並存し、どれに何があるか把握しづらい状態になっていると推測される。

## 2. オーナーの要望（再設計の軸）

タブ構成をドメイン（食事/生活）ではなく、**見せ方の種類**で再編してほしいとのこと。要望は3タブ構成。

### 2.1 折れ線グラフタブ

**全てのグラフを並べて見せる**。既存の`AnalyzeTab`（体重・カロリー・PFC・糖質/繊維/塩・歩数・睡眠・水分・筋トレ・カスタム習慣、それぞれ折れ線/棒グラフ）が実装としては近いが、現状は「体重・生活」タブの中に埋め込まれている。これを独立した「折れ線グラフ」タブとして切り出してほしい、という理解。

### 2.2 週間パネルタブ

摂取カロリーを**1週間の合計**で見たい。「1日2000kcalが目標なら週14000kcal、今何%摂取しているか」が分かること。表示上の要件:

- 横のバーは「1日の目安」として7分割に区切る
- ただし7分割はあくまで目安の区切りであり、**カロリーの合計（実績÷週目標の割合）をそのまま連続的に色付けする**（7分割の区切りを無視して塗る）

→ この仕様は直近の追加要望で会員側`WeeklySummaryPanel`にすでに実装済み（7ブロックはグリッドの目安線のみ、色塗りは`実績÷週目標`の連続値）。トレーナー向け（管理者側）でも同じ挙動を「週間パネル」という単独タブとして提供したい、という要望と理解している。

### 2.3 体重タブ（新規実装）

- 今週の平均体重 と 先週の平均体重 を比較
- 先週比で**何kg減ったか**、**減少率(%)** が分かること
- → これは既存の`WeightWeeklyCompare`（今週平均・先週比kg・先週比%）とほぼ同じ仕様。新規タブとして独立させたい、という要望
- 「体重のグラフだけはここにもあっていいかも」とのコメントあり（2.1の折れ線グラフタブと体重タブで、体重推移グラフが重複して2箇所に出ても構わない、という所感）

### 2.4 その他

オーナーの発言はここで一旦途切れている（「・」のみ）。追加の要望がまだ出てくる可能性がある。

## 3. Fable5への相談事項

1. 上記3タブ構成（折れ線グラフ／週間パネル／体重）への再編を採用するか。採用する場合、現行の「食事・栄養」「体重・生活」「週間まとめ」「目標設定」というドメイン別タブとどう統合・置き換えるか（目標設定タブは残す前提でよいか）
2. 「週間パネル」タブは、既存の食事・栄養項目（P/F/C/糖質/繊維/塩/歩数/水分/睡眠/筋トレ）も同じ「7分割目安＋連続塗り」の形式で並べるのか、カロリーだけを主役にするのか
3. 「体重タブ」に体重推移グラフ（2.1と重複）を含めるかどうか
4. 会員側（`WeeklyTab`の「詳細」「週間まとめ」切替、`AnalyzeTab`）にも同じ再編を適用するか、管理者側だけの再編に留めるか（F章以来の原則により、管理者側だけ整理して会員側据え置きにはできないため、必ず判断してほしい）

## 4. Sonnetからの追加提案（「もっと良い機能があれば」への回答）

- **今週のハイライト（自動要約）**: タブの最上部に「今週は先週より体重-0.4kg、カロリーは週目標比92%」のような1〜2行の自動生成サマリーを置く。数字を1つずつ追わなくても状態を把握できる
- **記録の抜け漏れの可視化**: 「週間パネル」に、記録できていない日を強調する簡易カレンダー的表示を添える（トレーナーが「最近記録が止まっている会員」に気づきやすくなる）
- **連続達成日数（ストリーク）**: 「今日の記録」が何日連続で続いているかを表示し、会員のモチベーション訴求にも使える指標としてトレーナー画面にも出す
- **グラフタブの期間切替**: 折れ線グラフタブに「週/月」の期間切替を持たせ、短期の変動と中期のトレンドを両方追えるようにする
- **体重タブへの目標体重ライン追加**: 体重推移グラフに目標体重の水平線を重ねると、トレーナーが進捗を一目で把握しやすくなる

これらは決定事項ではなく提案。採用の要否含めFable5の判断を仰ぎたい。
